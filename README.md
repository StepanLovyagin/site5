# Практическое занятие №5 — Проработка мобильного интерфейса Avion

Данная работа посвящена оптимизации пользовательского интерфейса сайта Avion для мобильных устройств в соответствии с международными гайдлайнами **Apple HIG** и **Google Material Design**.

## Что реализовано:

- **Адаптивное меню (Шаг 2):** Реализовано полноценное «выезжающее» меню (Drawer). 
  - Гамбургер-кнопка с плавной анимацией превращения в крестик.
  - Создание интерактивного оверлея через JavaScript.
  - Блокировка прокрутки основной страницы при открытом меню для удобства навигации.
- **Оптимизация Touch-интерфейса (Шаг 2):** 
  - Все интерактивные элементы (кнопки, ссылки) приведены к стандарту touch-зон — минимум **44x44px**.
  - Реализован визуальный **Feedback**: кнопки масштабируются (scale) при нажатии, имитируя физический отклик.
- **Типографика по гайдлайнам (Шаг 3):**
  - Размер основного текста установлен на **16px** (рекомендация Google Material Design).
  - Размер навигационных ссылок установлен на **17px** (рекомендация Apple HIG для iOS).
- **Sticky-навигация (Шаг 4):** Шапка сайта зафиксирована в верхней части экрана (`position: sticky`), что обеспечивает постоянный доступ к меню при скролле.
- **Кнопка «Наверх» (Шаг 5):**
  - Появляется автоматически при прокрутке страницы вниз более чем на 300px.
  - Реализована плавная прокрутка (Smooth Scroll) к началу страницы.
  - Размер кнопки (56px) оптимизирован для быстрого нажатия большим пальцем.

## Технологии и стандарты:
- **Методология:** БЭМ (Блок-Элемент-Модификатор).
- **Гайдлайны:** Apple Human Interface Guidelines, Google Material Design.
- **Доступность:** Соблюдение стандартов WCAG (размеры шрифтов, контрастность).
- **Стек:** HTML5, CSS3 (Mobile First / Desktop First медиазапросы), JavaScript.

## Использованные Breakpoints:
1. **1199px** — Ноутбуки (оптимизация отступов).
2. **1024px** — Планшеты (активация мобильного меню и гамбургера).
3. **768px** — Мобильные (альбомная ориентация, фиксация меню).
4. **480px** — Смартфоны (меню на всю ширину экрана, увеличение шрифта до 17px по HIG).

## Структура проекта:
- `index.html` — семантическая разметка.
- `assets/styles/style.css` — основные стили и десктопная версия.
- `assets/styles/mobile.css` — логика мобильного интерфейса.
- `assets/scripts/menu.js` — интерактивные функции (меню, кнопка скролла).
